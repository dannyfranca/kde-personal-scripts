# Shortcut Map

This file documents intended shortcuts for recovery on a fresh KDE Plasma install.

| Action | Intended shortcut | Notes |
|---|---|---|
| Smart Meta Up | `Meta+Up` | Appears under `System Services > Window Management` after the KWin script loads. Open that component first, then search for `Tile top`; KDE may not match the internal id `Smart Meta Up` in the top-level shortcut search. Conflicts with KDE's default `Quick Tile Window to the Top`; assign `Meta+Up` to `Smart Meta Up` and remove/reassign the built-in conflict. Side-half windows still move to top quarters; common top tiles maximize; script-created maximizes restore on the next press. |
